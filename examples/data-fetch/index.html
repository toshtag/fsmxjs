Data Fetch

A fetch that can fail. The machine owns every state — idle, loading, success, error — and a guard prevents retrying past the limit. No scattered flags, no isLoading/isError booleans, no useEffect deciding what happens next.

🔒 Auth flows: unauthenticated → authenticating → authenticated
