Reentrant Event Safety

Queue Mode

Without queue: true, calling send() from inside a subscriber is silently dropped — the machine is mid-transition and can't accept events. With queue mode enabled, those sends are enqueued and drained in order once the current transition finishes. Press the button and watch the event chain unfold.
